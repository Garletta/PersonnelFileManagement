# Personnel File Management
常州大学人事档案管理系统  

## 开发工具以及使用技术
IntelliJ IDEA开发工具
JAVA编程语言  
Spring框架、SpringMVC框架、Mybatis框架  
Tomcat网页部署服务器  
MariaDB开源数据库  
Bootstrap前端开发框架
